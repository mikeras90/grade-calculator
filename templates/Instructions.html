User Guide

This guide explains the complete workflow for using the Grade Calculator and Anonymizer Tool.

Part 1: First-Time Setup for a New Class

You only need to do this once per class at the beginning of the semester.

Step 1.1: Use the Anonymizer Tool

  1. Navigate to the anonymizer/anonymizer.html file on your computer and open it in a web browser.
  2. For the first input, select your **raw class roster** (CSV file).
  3. For the second input, select your **Week 1 transcript**.
  4. Click "Process Files". If there are unrecognized names, map them as needed.
  5. Your browser will download two files.
IMPORTANT: Save the anonymized_key.csv file in a safe place! This is your permanent "master key" for this class. You will need it every week.

Step 1.2: Set Up the Class in the Main App

  1. Go to the main application homepage.
  2. Create your new class.
  3. On the class page, you'll be prompted to create a roster. Enter the total number of students in your class and click "Create Roster".
  4. The page will now show a list of pseudonyms (Student-1, Student-2, etc.).
  5. Under "Transcript Analysis for Week 1", upload the anonymized_transcript.txt file you downloaded.

Part 2: Weekly Workflow (for Weeks 2+)

This is the process you will repeat each week.

Step 2.1: Use the Anonymizer Tool

  1. Open the anonymizer.html file again.
  2. For the first input, select your saved anonymized_key.csv file.
  3. For the second input, select the new week's transcript (e.g., Week 2 transcript).
  4. Click "Process Files".
  5. Your browser will download a new anonymized_transcript.txt for the current week.

Step 2.2: Upload Data to the Main App

  1. In the main application, navigate to the correct week using the week navigation bar (Week 2, Week 3, etc.).
  2. Enter and save any attendance data for that week.
  3. Upload the new anonymized_transcript.txt file for that week.

Part 3: Viewing Reports

At any time, you can use the links at the top of a class page to:

  • View Full Summary Report: See a grid of all data for all students across all weeks.
  • Grade Settings and Final Report: Adjust grading weights and view the final calculated grades for the semester.

Remember, the reports in the app will only show pseudonyms. Use your saved anonymized_key.csv file to match them to your real students.
